Introducción al lenguaje

HTML es un lenguaje que se usa para crear y organizar páginas web. Sirve para poner títulos, textos, imágenes y enlaces usando etiquetas.

Etiquetas de Formato

Br sirve para hacer un salto de línea. Hr sirve para dibujar una línea horizontal que separa contenido.
